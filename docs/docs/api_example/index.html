Rekor API Client Example

We also use OpenAPI specification in rekor making it easy to bootstrap your own RestFul API client.

If you’re interested in developing an application that can store or retrive entries into a Rekor transparency log, we recommend you take a look at the examples repository. In here you will find an example that downloads and stores the entire node.js signed release archive. It then monitors for new releases and automatically sends the signing artifacts to a rekor server.
